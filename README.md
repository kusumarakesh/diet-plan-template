## üöÄ Quick Start

### Development

```bash
# Install dependencies
npm install

# Start development server (http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Deployment

```bash
# Deploy to Firebase (requires initial Firebase setup)
npm run deploy

# Firebase authentication (one-time setup)
npm run firebase:login

# Initialize Firebase hosting (one-time setup)
npm run firebase:init
```
